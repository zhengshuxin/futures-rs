An I/O object which amortizes a number of read operations by reading many bytes at once into an internal buffer.

This object is intended to mirror the BufReader abstraction in the standard library and simply implements a stream of readiness on top of it as well.

fn into_inner(self) -> R

Consumes this object, returning the underlying I/O object.

Note that this discards the internal buffer of bytes, so it is recommended to read this reader until EOF if that is not desired.
