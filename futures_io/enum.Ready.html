pub enum Ready {
    Read,
    Write,
    ReadWrite,
}

Readiness notifications that a stream can deliver.

This is the primary point element yielded in Stream implementations on I/O objects, indicating whether it is ready to be read or written to.

Variants

Read

The I/O object is ready for a read.

Write

The I/O object is ready for a write.

ReadWrite

The I/O object is ready for both reading and writing.
